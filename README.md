# CMPS2212-Test1-CineSearchPro

Youtube Link: https://youtu.be/8dy6z3R0-3k?si=VpLxyTiJjrTe_hyV

A movie search dashboard, it allows users to search for movies using TMDB database api, you can browse search results, and view detailed movie information. In this project I used asynchronous data fetching, performance optimization, UI state management, and safe DOM manipulation. It includes debounce, request cancellation, caching, concurrent fetches, and XSS-safe rendering.  

Features:
Fast user typing without overloading the API
Cancelling outdated requests
Caching repeated searches for performance
Loading multiple pieces of movie data at the same time
Gracefully handling partial API failures
Rendering efficiently with minimal DOM updates
Providing keyboard accessibility
Avoiding unsafe HTML injection XSS vulnerabilities
